### 📌 Descripción  
Este proyecto es una versión de **WhatsApp Web**, desarrollada como parte del challenge de **Rolling Code School**. Se centra en la implementación de una interfaz amigable y fluida, simulando la experiencia de usuario de la aplicación original.  

🔗 **Demo en vivo:** https://challengerollingwhats.netlify.app/

---

## ⚙️ Tecnologías utilizadas  
- **HTML y CSS** - Desarrollo de la interfaz  
- **Styled Components / Boostrap CSS** - Estilos modernos y reutilizables
